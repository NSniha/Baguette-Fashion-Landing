# Baguette — Creative Design Studio Landing Page

A modern, editorial-style landing page for a creative design & branding studio, built with HTML, CSS, and vanilla JavaScript. Features bold serif typography, a scrolling announcement bar, and a visual portfolio grid — ideal as a starting template for branding agencies, packaging studios, and fashion/lifestyle brand portfolios.

## Features
- Scrolling announcement marquee
- Editorial-style hero section with bold serif typography
- Sticky navigation (Studio, Story, Services, Work, For Brands, Contact)
- Image-first portfolio/work grid
- Fully responsive
- Pure HTML/CSS/JS — no dependencies or build step

## Tech Stack
HTML5 · CSS3 · JavaScript (Vanilla)

## Getting Started
```bash
git clone https://github.com/NSniha/Baguette-Fashion-Landing.git
cd Baguette-Fashion-Landing
```
Open `index.html` in your browser, or serve it with a live server extension.
